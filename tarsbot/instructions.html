LDA - Load Accumulator

Loads a byte into the A registry.

V Overflow Flag Set if A > 0xFF
Z Zero Flag Set if A = 0

BUT - Ban User Temporarily

Bans user temporarily. User is specified with their ID at the given memory address. User IDs are 64 bit (8 bytes) unsigned numeric values. Length of ban determined by A registry (Days).

V Overflow Flag Not affected
Z Zero Flag Set if user was not found

BUP - Ban User Permanently

Bans user permanently. User is specified with their ID at the given memory address. User IDs are 64 bit (8 bytes) unsigned numeric values.

V Overflow Flag Set if A > 0xFF
Z Zero Flag Set if A = 0

PHA - Print Hex A

Prints the A registry to the current channel formatted in hex.

V Overflow Flag Not affected
Z Zero Flag Not affected

PDA - Print Decimal A

Prints the A registry to the current channel formatted in decimal.

V Overflow Flag Not affected
Z Zero Flag Not affected

PCA - Print Char A

Encode the byte in the A registry (UTF8) into a char and send it to the current channel. Because the A registry is only a single byte and unicode characters are 2 bytes, you can only print characters that range from 0x0000 to 0x00FF. If you want to print the full range of characters, please refer to PCM.

V Overflow Flag Not affected
Z Zero Flag Not affected
